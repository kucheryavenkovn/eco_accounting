Отчет "Занятость площадей"

Отчет предназначен для отображения графической информации по размещению объектов недвижимости на основе планов размещения в формате MS Excel. Объекты выделяются цветом в зависимости от того, какой статус они имеют на дату формирования отчета.

Состав текстовой информации для вывода на план может настраиваться произвольным образом. Для этих целей в форме отчета предусмотрена закладка Состав информации по объектам для вывода в отчет.

Текстовая информация по объектам может выводиться как внутри объекта, так и во всплывающих окнах, появляющихся при клике мышью по объекту. Во втором случае в форме отчета необходимо установить флаг Выводить информацию во всплывающих окнах. При этом дополнительно внутри объекта можно вывести его краткое представление: для этого в форме отчета нужно заполнить реквизит Краткое представление объекта.

Для оперативного перехода от объекта на плане к его карточке в базе 1С нужно кликнуть по объекту мышью. Если используется режим вывода текстовой информации во всплывающих окнах, то по объекту нужно кликнуть два раза: при первом клике появится всплывающее окно с информацией по объекту, при втором – предложение открыть карточку объекта.

С помощью отборов в отчете можно установить фильтр по заданным параметрам. Например, получить план с отображением только свободных объектов недвижимости или вывести объекты, занятые конкретным арендатором.

Настройка зависимости цвета отображения объекта на плане от его статуса выполняется в форме отчета по кнопке Настройка цветов.В сформированном отчете расшифровка значений цветов приводится на отдельном листе Легенда.
