Отчет "Структура расчетов по аренде"

Отчет предназначен для отображения информации о состоянии взаиморасчетов с арендаторами.

Отчет предусматривает возможность анализа взаиморасчетов с детализацией по договорам, периодам оплаты, услугам и объектам аренды.

В отчете выделены отдельные группы показателей для каждого вида задолженности:

  • Основная сумма (для расчетов по услугам аренды);
  • Пени (для расчетов по пеням);
  • Депозит (для расчетов по депозиту).

Для вывода суммарных показателей по основной сумме и пеням предусмотрена группа полей Общая сумма.

Для анализа расчетов по депозиту предусмотрен вывод различных показателей:

  • остаток оплаченного депозита на начало и конец периода отчета;
  • сумма оплаты депозита за период;
  • сумма депозита, зачтенного в счет оплаты услуг;
  • размер депозита по условиям договора на конец периода отчета;
  • сумма депозита к оплате с учетом условий договора и остатка депозита;
  • итоговые показатели расчетов по договору с учетом данных по депозиту.

Предусмотрена возможность расшифровки данных, позволяющая развернуть информацию до документа–регистратора.
