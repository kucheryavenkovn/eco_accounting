Отчет "План-фактный анализ поступления денежных средств "

 

Отчет предназначен для анализа исполнения планов по поступлению денежных средств за выбранный период.

 

Поле Периодичность используется для приведения как плановых, так и фактических данных к одному значению периодичности.

 

Параметр Период оплаты для фактических поступлений определяет логику вычисления поля Период оплаты для фактических платежей: по дате документа или по периоду аренды. Второй вариант имеет смысл только в случае ведения взаиморасчетов по договору по периодам аренды. Если выбран второй вариант, но взаиморасчеты по договору по периодам аренды не ведутся, то период оплаты будет определен по дате документа.

 

Если указан период отчета, то в отчет попадут только те фактические платежи, у которых дата документа оплаты находится внутри периода отчета.

 

Если требуется отобрать данные по полю Период оплаты, то для этого в настройках отчета необходимо установить соответствующий отбор.

 

В отчете предусмотрена возможность расшифровки данных, позволяющая развернуть информацию до документа–регистратора.
