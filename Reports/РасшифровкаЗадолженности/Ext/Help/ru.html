Отчет "Расшифровка задолженности"

Отчет предназначен для формирования расшифровки по результатам инвентаризации расчетов с дебиторами и кредиторами.

Если на дату формирования отчета есть проведенные акты инвентаризации расчетов с детализацией по срокам, то информация в отчет выбирается из актов инвентаризации, иначе - из данных бухгалтерского учета. Сверить данные актов инвентаризации и бухгалтерского учета можно, включив флажок Контроль по данным бухучета в настройках отчета.

Отчет может быть сформирован с детализацией:

  • по срокам возникновения задолженности;
  • по срокам погашения задолженности;
  • без детализации по срокам.

В случае формирования отчета по срокам возникновения или погашения задолженности на закладке Настройка интервалов указываются периоды группировки задолженности по срокам (до 90 дней, от 90 до 180 дней и т.д.).

В отчете предусмотрена группировка данных:

  • по видам задолженности - дебиторская и кредиторская в зависимости от признака "Активный/Пассивный" счета, на котором учитывается задолженность;
  • по строкам бухгалтерского баланса - 1170 "Финансовые вложения", 1230 "Дебиторская задолженность", 1520 "Кредиторская задолженность" и др. Итоги по срокам баланса выводятся в полной сумме для всех входящих в строку баланса счетов бухгалтерского учета. При этом данные по счетам, которые не участвуют в актах инвентаризации расчетов с дебиторами и кредиторами, отображаются сводно (без детализации по аналитике).
