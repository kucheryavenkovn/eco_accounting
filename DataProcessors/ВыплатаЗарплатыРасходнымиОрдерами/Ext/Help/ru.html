Выдача наличных на каждого сотрудника

Обработка предназначена для автоматического формирования кассовых документов для выдачи зарплаты или доходов по договорам подряда сотрудникам.

В поле Платежная ведомость следует указать документ, на основании которого производится выплата. Табличная часть заполнится автоматически.

Сформировать список кассовых документов по отмеченным сотрудникам можно по кнопке Создать документы.

Провести созданные документы Выдача наличных можно по кнопке Провести документы.

По одной ведомости создавать и проводить кассовые документы можно в разное время. В этом случае уже созданные по документу кассовые документы будут автоматически найдены и заполнены в списке.
