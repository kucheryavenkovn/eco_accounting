Обработка "Групповое начисление арендной платы"

 

Обработка предназначена для группового формирования документов Начисление арендной платы за выбранный период.

Если установлен флаг Начислять постоянную часть оплаты, то в состав формируемых документов будут включены услуги, относящиеся к постоянной части арендной платы (услуги с установленным признаком Постоянная часть арендной платы). Если установлен флаг Начислять переменную часть оплаты, то в состав формируемых документов будут добавлены услуги, относящиеся к переменной части арендной платы.

В форме обработки можно выбрать способ заполнения документов – По счетам или Рассчитать. Если выбран способ По счетам, то документы начисления арендной платы будут сформированы на основании созданных ранее счетов на аренду. Этот способ удобен, например, в тех случаях, когда в счета после их автоматического формирования вручную вносились какие–либо изменения. Если выбран способ Рассчитать, то документы начисления арендной платы будут заполнены по данным действующих договоров аренды и документов Регистрация объемов потребления услуг переменной части за соответствующий период (аналогично обработке группового выставления счетов).

С помощью реквизита Вариант учета НДС при формировании документов по услугам переменной части можно использовать вариант учета НДС, отличный от указанного в договоре. Например, когда в договоре указан вариант НДС в сумме, но для каких-либо услуг переменной части требуется использовать вариант НДС сверху.

В группе Время записи документов выбирается один из вариантов установки времени записи для создаваемых документов. Возможные значения: Текущее время, Начало дня, Конец дня

Если за выбранный период по данным отбора уже имеются документы начисления аренды, то для таких случаев пользователь может выбрать один из следующих вариантов действий:

  • Пометить на удаление все ранее сформированные документы;

  • Отменить формирование всех новых документов – в случае, если в выбранном периоде уже есть хотя бы один документ, будет отменено формирование всех новых документов и выдано соответствующее предупреждение;

  • Оставить ранее сформированные, повторные не создавать – если в выбранном периоде уже есть документы по данному арендатору, объекту, услуге и т.п., то будут оставлены существующие документы, повторные документы создаваться не будут. Данный способ удобно использовать, например, когда для части арендаторов документы были созданы ранее, а позже необходимо создать документы для всех остальных арендаторов;

  • Создать повторно – документы будут созданы независимо от того, создавались они ранее за выбранный период или нет.

Закладка Отбор данных позволяет ограничить количество и состав формируемых документов с помощью отбора по различным аналитикам:

  • Организация;

  • Контрагент;

  • Договор;

  • Объект аренды;

  • Услуга.

Если отбор не указан, то документы будут формироваться по всем арендаторам, услугам, объектам аренды.

Для создания документов необходимо нажать кнопку Сформировать документы.

Все созданные документы отобразятся на закладке Сформированные документы, где их можно распечатать (с предварительным просмотром или сразу на принтер) и провести в пакетном режиме. Предусмотрена возможность отправки документов по электронной почте.

Кнопка Выставление СФ позволяет на основании сформированных документов автоматически создать документы Счет-фактура выданный.
