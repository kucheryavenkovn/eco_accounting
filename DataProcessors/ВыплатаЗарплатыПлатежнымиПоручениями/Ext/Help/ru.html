Платежные поручения на каждого сотрудника

Обработка предназначена для автоматического формирования платежных поручений для перечисления зарплаты или доходов по договорам подряда сотрудникам.

В поле Платежная ведомость следует указать документ, на основании которого производится перечисление. Табличная часть заполнится автоматически.

Сформировать список платежных поручений по отмеченным сотрудникам можно по кнопке Создать документы.

Отправить  созданные документы Платежное поручение  в банк можно по кнопке Выгрузить документы.

По одной ведомости создавать и отправлять платежные поручения можно в разное время. В этом случае уже созданные по документу платежные поручения будут автоматически найдены и заполнены в списке.
