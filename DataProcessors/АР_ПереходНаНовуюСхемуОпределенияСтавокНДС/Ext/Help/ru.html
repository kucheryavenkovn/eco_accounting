Обработка "Переход на схему определения ставок НДС по услугам аренды на основе договора"

Обработка позволяет включить схему определения ставок НДС по услугам аренды на основе договора и заполнить параметры учета НДС в договорах аренды (документы Заключение договора аренды, Дополнительное соглашение, Расторжение договора аренды). Порядок использования:

  1. Включить схему определения ставок НДС по услугам аренды на основе договора, установив соответствующий флаг. При этом в документах Заключение договора аренды, Дополнительное соглашение и Расторжение договора аренды появится возможность выбора варианта учета НДС, а в табличных частях появится возможность выбора ставки НДС по услугам.
  2. Установить условия отбора документов, для которых требуется заполнить параметры НДС. Например, можно установить отбор по организации, дате документа (чтобы не вносить изменения в договоры аренды, которые уже не действуют) и прочим реквизитам документов.
  3. Заполнить список документов нажатием соответствующей кнопки. В список попадут только те документы, у которых есть хотя бы одна услуга (постоянной или переменной части) с незаполненной ставкой НДС (и при этом в документе не установлен признак Документ без НДС).
  4. На закладке Параметры НДС выбрать нужный вариант учета НДС и заполнить список услуг и ставок НДС. На основе этих данных будут заполняться параметры НДС в документах.
  5. Нажать кнопку Выполнить. В результате в документах из списка будут заполнены параметры учета НДС в соответствии с указанными настройками.

При необходимости обработку можно запускать многократно. Например, сначала установить параметры НДС в договорах аренды одной организации, а потом в другой.

При снятии флага Включить схему определения ставок НДС по услугам аренды на основе договора сохраняются все изменения, внесенные обработкой в документы по договорам аренды. Это позволяет при необходимости отключить новую схему определения НДС, чтобы выполнить расчет аренды по старой схеме, после чего продолжить заполнение параметров НДС в договорах аренды для перехода на новую схему.
