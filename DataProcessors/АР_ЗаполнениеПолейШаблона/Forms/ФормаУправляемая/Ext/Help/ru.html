Справочник "Шаблоны документов"


Справочник содержит список шаблонов формата MS Word для формирования печатных форм документов. Редактирование шаблона и размещение в нем полей для автоматического заполнения осуществляется в обработке Заполнение полей шаблона MS Word, которая вызывается двойным кликом мышью по названию шаблона.
