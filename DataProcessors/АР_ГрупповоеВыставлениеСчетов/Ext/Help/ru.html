Обработка "Групповое выставление счетов"

 

Обработка предназначена для группового выставления счетов на оплату услуг по договорам аренды за выбранный период, а также для автоматического формирования счетов на оплату пеней. Для выбора нужной операции предназначен реквизит Вид операции.

Вид операции Оплата аренды

Если установлен флаг Постоянная часть оплаты, то в состав формируемых счетов будут включены услуги, относящиеся к постоянной части арендной платы (услуги с установленным признаком Постоянная часть арендной платы). Информация для заполнения берется из условий договоров аренды, действующих в выбранном периоде начисления. Если установлен флаг Переменная часть оплаты, то в состав формируемых счетов будут добавлены услуги, относящиеся к переменной части арендной платы. Информация для заполнения берется из документов Регистрация объемов потребления услуг переменной части за соответствующий период.

С помощью реквизита Вариант учета НДС при формировании документов по услугам переменной части можно использовать вариант учета НДС, отличный от указанного в договоре. Например, когда в договоре указан вариант НДС в сумме, но для каких-либо услуг переменной части требуется использовать вариант НДС сверху.

Если за выбранный период по данным отбора уже имеются выставленные счета на оплату аренды, то для таких случаев пользователь может выбрать один из следующих вариантов действий:

  • Пометить на удаление все ранее сформированные документы;

  • Отменить формирование всех новых документов – в случае, если в выбранном периоде уже есть хотя бы один документ, будет отменено формирование всех новых документов и выдано соответствующее предупреждение;

  • Оставить ранее сформированные, повторные не создавать – если в выбранном периоде уже есть документы по данному арендатору, объекту, услуге и т.п., то будут оставлены существующие документы, повторные документы создаваться не будут. Данный способ удобно использовать, например, когда для части арендаторов документы были созданы ранее, а позже необходимо создать документы для всех остальных арендаторов;

  • Создать повторно – документы будут созданы независимо от того, создавались они ранее за выбранный период или нет.

Вид операции Оплата пени

Выполняется автоматическое формирование счетов на оплату пеней.

Алгоритм заполнения документов аналогичен алгоритму расчета пеней по кнопке Заполнить в форме счета.

Закладка Отбор данных позволяет ограничить количество и состав формируемых документов с помощью отбора по различным аналитикам:

  • Организация;

  • Контрагент;

  • Договор;

  • Объект аренды;

  • Услуга.

Если отбор не указан, то счета будут формироваться по всем арендаторам, услугам, объектам аренды.

Для создания документов необходимо нажать кнопку Сформировать документы.

Созданные документы отобразятся на закладке Сформированные документы. По кнопке Печать документы могут быть распечатаны в пакетном режиме: с предварительным просмотром или сразу на принтер. Предусмотрена возможность отправки документов по электронной почте.
