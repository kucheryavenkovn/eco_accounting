Обработка "Групповое оформление заявок на обслуживание"

 

Обработка предназначена для пакетного формирования документов «Заявка на обслуживание» по данным о графиках обслуживания объектов недвижимости и их элементов.

В шапке обработки пользователь должен указать период, в котором планируется проводить работы по обслуживанию. Этот период будет учитываться при анализе графиков обслуживания объектов недвижимости.

 

На закладке «Отбор данных» пользователь может указать необходимые отборы по объектам недвижимости, элементам, комплексам работ и работам, спецификациям, организациям и др.

 

На закладке «Сформированные документы» отображается список сформированных заявок на обслуживание.

 

Для формирования заявок необходимо нажать кнопку «Сформировать».
