Обработка "Групповое выставление счетов фактур"

 

Обработка предназначена для группового формирования документов «Счет фактура выданный» на основании документов «Начисление арендной платы» за выбранный период.

 

На форме обработки располагается поле для отбора арендаторов. Только выбранным арендаторам будут выставлены счета фактуры.

 

Для формирования документов необходимо нажать кнопку «Сформировать документы».
