Документ "Регистрация объемов потребления услуг переменной части"

 

Документ предназначен для регистрации объемов потребления и расчета стоимости услуг, относящихся к переменной части арендной платы. Документ позволяет учитывать как услуги, сумма которых рассчитывается по простой формуле умножением объема потребления на тариф, так и услуги, сумма которых вычисляется по произвольным формулам и при выполнении произвольных условий – например, на основе процента от торгового оборота, если его величина превышает арендную плату.

 

Предусмотрено два способа заполнения документа, определяющих его внешний вид:

  • заполнение данных по всем услугам переменной части, указанным в конкретном договоре арендатора;
  • заполнение данных по всем договорам аренды, в которых указана конкретная услуга.

 

Для выбора способа заполнения в форме документа расположен переключатель Способ заполнения документа. Если установлено значение этого переключателя По выбранному договору арендатора найти все услуги, то реквизиты Арендатор и Договор располагаются в шапке документа, а реквизит Услуга – в табличной части. Если же установлено значение По выбранной услуге найти всех арендаторов, то реквизиты Арендатор и Договор располагаются в табличной части, а реквизит Услуга – в шапке документа.

 

Предусмотрено несколько вариантов автоматического заполнения табличной части документа:

  • По команде Заполнить по договорам аренды. Если выбран способ заполнения документа По выбранному договору арендатора найти все услуги, то заполнение табличной части выполняется на основе списка услуг, предусмотренных в указанном договоре. Если выбран способ заполнения документа По выбранной услуге найти всех арендаторов, то заполнение табличной части выполняется по всем договорам аренды, включающим указанную в шапке документа услугу. Если за выбранный период изменялась ставка (тариф) на услугу или арендуемая площадь, то в табличную часть добавляется столько строк, сколько раз происходили изменения;
  • По команде Заполнить по договорам аренды с отбором происходит заполнение табличной части по алгоритму, аналогичному предыдущему способу, но с возможностью отбора по заданным критериям (например, по арендаторам, объектам и т.д.) Кнопка Заполнить по договорам аренды с отбором доступна только для способа заполнения документа По выбранной услуге найти всех арендаторов;
  • Команда Заполнить по зданию предназначена для обеспечения возможности распределения затрат по коммунальным услугам пропорционально площадям с учетом общей площади здания, включая как арендуемые, так и неарендуемые площади. По этой команде заполнение происходит на основе действующих договоров аренды, а в конце добавляется строка с разностью общей площади здания и площадей в аренде.
    Если какой-либо объект находился в аренде не полный период, то для него при заполнении по данной команде колонка Площадь по договору пересчитывается пропорционально количеству дней, которые объект был в аренде.
    Дальнейшее распределение показаний по всем площадям выполняется выбором соответствующего способа расчета (см. далее);
  • Команда Рассчитать суммы по формулам позволяет рассчитать итоговые суммы в строках документа на основании введенных показаний, если для услуги установлен вариант расчета по произвольной формуле (см. далее);
  • Команда Распределить показания счетчиков позволяет заполнить табличную часть документа, распределив показания счетчиков между арендаторами в соответствии с коэффициентами отнесения показаний счетчиков, прописанных в договоре аренды. Этот вариант удобен тем, что пользователю достаточно внести в табличную часть только список счетчиков и данные их показаний на начало и конец периода. Распределение показаний по арендаторам и расчет сумм программа выполнит автоматически;
  • По команде Установить тариф можно изменить значение тарифа одновременно во всех строках документа с автоматическим пересчетом итоговых сумм.

 

Документ Регистрация объемов потребления услуг переменной части может вводиться на основании другого документа Регистрация объемов потребления услуг переменной части, относящегося к предыдущему периоду. В этом случае табличная часть документа будет заполнена так же, как и табличная часть документа-основания, и при этом показания на начало периода будут заполнены показаниями на конец периода, указанными в документе-основании, а показания на конец периода будут обнулены, т.к. для них необходимо ввести актуальные значения.

 

Предусмотрена возможность заполнения показаний на основе распределения общей величины показаний. Также предусмотрена возможность распределения по строкам документа общей стоимости услуги. Для выбора варианта распределения предназначен реквизит Ввод показаний, предусматривающий следующие значения выбора:

  • Фиксированной суммой (или пустое значение) – предполагает ручное заполнение показаний;
  • Показание пропорционально площади объектов позволяет указать общую величину показаний для выбранной услуги и распределить ее по строкам табличной части документа пропорционально арендуемым площадям;
  • Сумма пропорционально площади объектов – позволяет указать общую стоимость услуги и распределить ее по строкам табличной части документа пропорционально арендуемым площадям;
  • Сумма равномерно по договорам – позволяет указать общую стоимость услуги и распределить ее равномерно по строкам табличной части документа.

 

Способ расчета суммы по показаниям потребления, указанным в строке документа, определяется в соответствии со способом начисления, заданным для данной услуги в документе заключения договора аренды. Как правило, сумма вычисляется по формуле:

 

Сумма = Разница показаний * Тариф

 

Если способ начисления предусматривает расчет по произвольной формуле, то сумма вычисляется в соответствии с установленной формулой.

 

Если при заключении договора аренды для услуги был установлен способ начисления, предусматривающий расчет по произвольной формуле, то при автоматическом заполнении табличной части документа для строк, содержащих данную услугу, в поле Сумма будет выводиться специальная пиктограмма.

 

В таких строках расчет суммы выполняется не сразу при изменении показаний или тарифа, а по отдельной команде. Для расчета суммы в конкретной строке предназначена команда Рассчитать по формуле, вызываемая из контекстного меню строки. Для расчета сумм во всех строках табличной части предназначена команда Рассчитать суммы по формулам из подменю Заполнить.

 

Если услуга, указанная в документе, относится к услугам телефонии, и в ее карточке установлен соответствующий признак Услуги телефонной связи, то в табличной части документа будет отображаться колонка Номер телефона, с помощью которой можно детализировать расходы по телефонным номерам.

 

Если требуется зафиксировать объемы потребления услуг, которые не подлежат выставлению арендаторам к оплате, но которые должны быть учтены с целью их отражения в отчете по коммунальным услугам, то в табличную часть документа нужно добавить соответствующие строки, в которых поля для указания арендатора и договора оставить пустыми.
