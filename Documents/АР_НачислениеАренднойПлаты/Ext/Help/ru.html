Документ "Начисление арендной платы"

 

Этот документ является основным документом для начисления арендной платы за определенный период (формирования задолженности арендатора перед арендодателем).

 

Документ может быть введен на основании документа «Счет на аренду» с видом операции «Оплата аренды». В этом случае табличная часть заполняется данными из табличной части счета. Кроме этого, табличная часть может быть заполнена автоматически при помощи кнопок на командной панели.

 

Для того, чтобы заполнить табличную часть данными счетов на аренду, необходимо воспользоваться меню «Добавить по счету». Для того, чтобы рассчитать начисления арендной платы без учета документов «Счет на аренду», необходимо воспользоваться меню «Рассчитать» (в этом случае табличная часть документа заполняется по аналогии с автозаполнением табличной части счета на аренду).

 

При проведении документа система анализирует состояние взаиморасчетов по арендным платежам. При этом анализируются все дополнительные аналитики, в разрезе которых ведутся взаиморасчеты (период оплаты, объекты аренды и услуги, счета на аренду).

 

В случае установленного флага «Зачет депозита», в качестве аванса может быть зачтен обеспечительный депозит. В случае, когда этот флаг сброшен, сумма обеспечительного депозита в качестве авансовых платежей не учитывается.

 

Также при проведении осуществляется контроль соответствия табличной части документа условиям договора аренды, прописанным в документе "Заключение договора аренды". Для пользователей, у которых установлена роль "Полные права" добавлена возможность отключения данного контроля. Для этого в документе на закладке "Дополнительно" необходимо установить признак "Не контролировать соответствие содержанию договора".

 

При использовании агентской схемы взаиморасчетов между организацией и собственником объектов недвижимости в строках табличной части документа необходимо заполнять колонки Принципал, Агентский договор и Счет расчета с принципалом. При автоматическом заполнении эти данные будут браться из регистра Собственники объектов недвижимости. При этом если для объекта указано несколько собственников и указаны признаки использования агентской схемы, то распределение суммы и количества между собственниками осуществляется пропорционально долям собственности. Для расчета агентского вознаграждения следует использовать типовой документ Отчет комитенту (принципалу) о продажах.

 

Документ является основанием для ввода документа «Счет-фактура выданный». Для быстрого ввода счета-фактуры можно воспользоваться гиперссылкой в подвале документа.
