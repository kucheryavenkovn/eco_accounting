Справочник "Типы объектов недвижимости"

 

Предназначен для хранения возможных типов объектов недвижимости (офис, склад, торговое помещение, административное помещение и пр.).

Реквизит "Номенклатурная группа" предназначен для заполнения соответствующей аналитики по бухгалтерскому учету при создании документов по аренде. Если данный реквизит не заполнен, то номенклатурная группа в документах будет заполняться по услуге.
