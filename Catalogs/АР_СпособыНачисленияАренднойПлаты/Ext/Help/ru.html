Справочник "Способы начисления арендной платы"

 

Справочник предназначен для хранения различных способов начисления оплаты за услуги (например, «В руб. за кв. метр/месяц»). 

 

В справочнике предусмотрены три варианта расчета: по ставке за единицу площади, по ставке за объект в целом и расчет по произвольной формуле. 

 

Общими для всех вариантов расчетов являются поля:

  • Валюта. Валюта начисления, определяет валюту ставки за услугу.
  • Период начисления. Период начисления, соответствующий ставке за услугу.

 

Вариант расчета По ставке за единицу площади предполагает, что ставка, указанная в документе, относится к единице арендуемой площади. 

 

Вариант расчета По ставке за объект в целом предполагает, что ставка, указанная в документе, относится ко всему объекту в целом, независимо от его площади. 

 

Вариант расчета Расчет по произвольной формуле предназначен для услуг, относящихся к переменной части арендной платы. Данный вариант позволяет указать произвольную формулу для расчета начислений. 

 

В качестве операндов в формуле могут использоваться произвольные числа, а также предопределенные переменные, указанные в левом окне в группе Операнды:

  • Арендная плата – вычисляется автоматически как начисление за услугу аренды недвижимости по данному объекту за период, указанный в документе Регистрация объемов потребления услуг переменной части;
  • Показатель – разница показаний на конец и начало периода, указанных в документе Регистрация объемов потребления услуг переменной части;
  • Ставка – значение поля Тариф в документе Регистрация объемов потребления услуг переменной части;
  • Площадь – значение поля Площадь по договору в документе Регистрация объемов потребления услуг переменной части.

 

Проверить корректность введенной формулы можно с помощью кнопки Проверить формулу. 

 

Расчет по произвольной формуле позволяет, в частности, выполнять расчет арендной платы на основе процента с торгового оборота. В этом случае формула расчета может быть задана, например, следующим образом: 

 

?(Показатель*Ставка > АренднаяПлата, Показатель*Ставка - АренднаяПлата, 0) 

 

Данная формула позволяет начислить арендатору разницу между процентом с торгового оборота и фиксированной частью арендной платы. Здесь Показатель – это торговый оборот по объекту, который нужно указать в документе Регистрация объемов потребления услуг переменной части в поле Показание на конец периода; Ставка – процент с торгового оборота, который изначально устанавливается в качестве ставки на услугу в документе Заключение договора аренды, а в документе Регистрация объемов потребления услуг переменной части выводится в поле Тариф. 

 

Подробнее о работе с документом Регистрация объемов потребления услуг переменной части см. в справке по этому документу.
