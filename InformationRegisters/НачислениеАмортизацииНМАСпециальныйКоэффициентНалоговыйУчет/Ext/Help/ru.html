Регистр сведений "Специальный коэффициент для амортизации НМА (налог. учет)"

В этом регистре хранятся значения понижающего коэффициента для начисления амортизации по налоговому учету согласно п. 10 статьи 259  НК РФ. Регистр периодический.

Влияет на начисление амортизации по объектам НМА в налоговом учете - значение специального коэффициента определяется на начало месяца начисления амортизации.

Запись в регистр производится документом "Принятие к учету НМА" - при вводе первоначальных сведений по объекту и документом "Изменение понижающего коэффициента для расчета амортизации НМА (налоговый учет)" - при изменении коэффициента для использования в следующем налоговом периоде.
